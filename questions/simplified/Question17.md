# Problem Statement

Given a list of coordinates representing red tiles in a grid, find the largest rectangle that can be formed with these tiles as opposite corners. The area of the rectangle is defined by the distance between the coordinates in the x and y directions.

## Input Format

A list of coordinates in the format x,y, where each coordinate represents the position of a red tile in the grid.

## Output Format

An integer representing the area of the largest rectangle that can be formed by selecting any two red tiles as opposite corners.

## Sample Input

```
7,1
11,1
11,7
9,7
9,5
2,5
2,3
7,3
```

## Sample Output

```
50
```
